Gestion des comptes Steam
Steam IDs autorisés
Le dashboard n'enregistre que les parties des Steam IDs listés ici. Ça évite que les données d'un joueur spectateur soient enregistrées.
Liste vide = tout le monde est accepté (mode setup, à éviter en jeu).

Trouve ton Steam ID sur steamidfinder.com ou dans CS2 → status dans la console.
⚠ Liste vide : toutes les données GSI sont acceptées, y compris celles des spectateurs. Ajoute au moins un Steam ID.
✓ Filtrage actif — seuls les Steam IDs listés sont enregistrés.
Aucun Steam ID configuré — toutes les données sont acceptées
Comment trouver son Steam ID
Méthode 1 : Lance CS2 → ouvre la console → tape status → cherche la ligne avec ton pseudo, le Steam ID est au format 76561198XXXXXXXXX

Méthode 2 : Va sur steamidfinder.com et entre ton pseudo Steam.

Méthode 3 : Dans Steam → clic droit sur ton profil → Copier l'URL du profil → colle sur steamidfinder.com.

Tu peux ajouter autant de comptes que tu veux.
